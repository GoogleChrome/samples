Background

All content that does not have the will-change: transform CSS property will be re-rastered when its transform scale changes. In other words, will-change: transform effectively means "please animate it fast" without taking the additional time for rasterization. This only applies to scaling that happen via JavaScript manipulation, and does not apply to CSS animations.
