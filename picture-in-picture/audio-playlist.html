Background

The Picture-in-Picture API allows websites to create a floating video window that is always on top of other windows so that users may continue consuming media while they interact with other sites or applications on their device. Combining this with HTMLCanvasElement.captureStream() and the Media Session API, web developers can create an audio playlist window.
