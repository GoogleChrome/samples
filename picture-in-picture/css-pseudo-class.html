Background

The Picture-in-Picture API allows websites to create a floating video window that is always on top of other windows so that users may continue consuming media while they interact with other sites or applications on their device. In Chrome 76, a CSS pseudo-class is added to help web developers customize the media player when videos enter and exit Picture-in-Picture.
