Background

The ResizeObserver API provides an API to get notified whenever an element changes its size.

This sample illustrates how to use ResizeObserver to implement [media query](https://developer.mozilla.org/en-US/docs/Web/CSS/Media_Queries/Using_media_queries) -like behavior on a per-element basis.

The Google Developers Web Updates article provides more details on ResizeObserver. As of Chrome 54, it is behind the `chrome://flags/#enable-experimental-web-platform-features` flag.
