Background

If you're seeing this page, you've most likely been navigated here from index.html following the service worker activation.

The service worker called client.navigate('activated.html') on each window that it controlled, which caused all of them to navigate to this page.

When you revisit index.html, you'll remain on that page, since the activate event is only fired once for each service worker version.
