Network Information Sample

Available in Chrome 38+ (Android, iOS, ChromeOS)

This demo illustrates the use of the Network Information API to retrieve the current network type and monitor for network type changes.

The Network Information API is not initially available on all platforms. For best results, try this demo using Chrome on Android, iOS, or ChromeOS.

On supported devices, changing your network connection will update the type on this page.
